# cat-finder
Finds cat faces in photos.

![cat](http://i.imgur.com/G6ZxwGO.png)

# Getting started

# 1. Dependencies
    sudo apt-get install python python-dev python-opencv

# 2. Clone repository
    cd ~
    git clone https://github.com/kendricktan/cat-finder.git
    cd cat-finder
    python cat.py
